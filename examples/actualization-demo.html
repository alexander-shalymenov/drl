What this means

  

    This demo makes one point clear: the most probable candidate is not always the final committed state.
    DRL balances probability with description complexity.
  

  

    For browser simplicity, K(ω) is approximated by string length in this demo.
    In the Node.js implementation, K(ω) can be approximated by compressed byte length.
